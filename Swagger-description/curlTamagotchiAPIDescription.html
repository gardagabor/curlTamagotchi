put /actions/drink/{id}
Increase the THIRSTINESS stat to 100 to a tamagotchi with a given ID (drinkTamagotchiById)

Path parameters

id (required)
Path Parameter — Numeric ID of the tamagotchi to drink

Responses

200

successful operation

405

Given ID is invalid

410

You are not in the KITCHEN

414

There is no WATER left

get /actions
Return all the possible actions in the game (getAllActions)
Return all the possible actions in the game with description

Return type

array[Action]

Example data

Content-Type: application/json
[ {
  "name" : "Move",
  "description" : "Move to another place, decrease ENERGY by 15"
}, {
  "name" : "Eat",
  "description" : "Increase the HUNGRINESS to 100, only possible if being in the KITCHEN and there are FOOD to consume"

  "name" : "Drink",

Rules

Up 
get /rules
Return the game rules (getRules)
Return the game rules and the losing conditions (there is no such as winning condition, it's a tamagotchi game)

Return type

String

Example data

Content-Type: application/json
"Welcome to the Curl Tamagotchi game, where you only lose. The point of the game is to keeping alive as much tamagotchis as you can. Each hour every tamagotchis stats(HUNGRINESS, THIRSTINESS, HAPPINESS, ENERGY) are decreasing by a random amount (10-30). To keep the tamagotchis alive you have to certain actions (MOVE, EAT,DRINK, PLAY, SLEEP). These actions increase the tamagotchis stats by a given number. If any of the stats goes below 0, your tamagotchi dies :(. More detailed descripions are under root endpoints."

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • text/plain

Responses

200

successfull operation String
